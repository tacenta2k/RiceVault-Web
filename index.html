✦ Core Features

Everything your rice needs.

A complete toolkit for managing, versioning, and sharing your KDE Plasma desktop configurations.

Git-Like Workflow

Commit snapshots of your entire Plasma setup. Branch, diff, and roll back your desktop configurations with familiar version control semantics.

rice commit rice diff rice log

Safe Restores

Restore any previous theme snapshot instantly. Auto-backup before every restore ensures you never lose your current setup, even when experimenting.

rice restore auto-backup

Blazing Fast CLI

Sub-second operations. Lightweight Python CLI with zero bloat — no GUIs, no daemons, no unnecessary dependencies. Just pure speed.

→ Backup completes in <200ms

Portable Profiles

Export your rice as a self-contained archive. Share it with friends, post it on r/unixporn, or transfer it to a new machine in seconds.

rice export rice import

⬡ Get Started

Installation

Up and running in under 30 seconds. Just clone and install.

bash 
# Clone the repository
$ git clone https://github.com/tacenta2k/RiceVault.git

# Navigate into the directory
$ cd RiceVault

# Install with pipx (recommended)
$ pipx install .

rice backup

Save your current rice

rice list

View saved profiles

rice restore

Restore any snapshot
